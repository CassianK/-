## 🎨 About This Project

**Art for Life**는 잊힌 기억과 음악을 통해  
삶의 균열을 조용히 복원해 가는 여정을 기록합니다.  
음악은 단지 듣는 것이 아니라,  
전생과 현생을 잇는 리듬입니다.
